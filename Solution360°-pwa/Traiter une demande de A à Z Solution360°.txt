🚀 WORKFLOW COMPLET D'UTILISATION
Scénario : Traiter une nouvelle demande
ÉTAPE 1 : Consulter la demande

Ouvrir le lien

Lire la description dans la colonne gauche

Vérifier le budget proposé

ÉTAPE 2 : Fixer le prix

Aller dans l'onglet "Tarification"

Saisir le prix final (ex: 50 000 FCFA)

Justifier (ex: "Projet complexe nécessitant 2 semaines de travail...")

Cliquer sur "Envoyer le devis au client"

✅ Le statut passe automatiquement à "En attente paiement"

ÉTAPE 3 : Envoyer une réponse

Aller dans l'onglet "Réponse"

Choisir un template ou rédiger

Cliquer sur "Envoyer la réponse au client"

ÉTAPE 4 : Changer le statut

Aller dans l'onglet "Statut"

Sélectionner "En production" (après paiement)

Indiquer la raison : "Paiement reçu, début des travaux"

Cliquer sur "Mettre à jour"

ÉTAPE 5 : Uploader les livrables

Aller dans l'onglet "Livrables"

Glisser-déposer vos fichiers finaux

Les fichiers sont stockés dans Supabase Storage

ÉTAPE 6 : Marquer comme livré

Retourner dans "Statut"

Changer vers "Livré"

Ajouter une note : "Projet terminé et livré"
